# E-Commerce WebSite
## Tech Stack
- Mongo DB
- React Js
- Node Js
- BrainTree (For payment)
##  How To Run The Site
-  Clone the repo
-  Open Terminal
### To Launch the front end (React Js)
- cd frontend
-  npm install
- npm start
### To start the server (Node Js)
- cd backend
- npm install
- npm test
