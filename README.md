# E-Commerce WebSite
>
## Tech Stack
- Mongo DB
- React JS
- Node JS
- BrainTree (For payment)
>
##  How To Open The Website and run the server
-  Clone the repo
-  Open Terminal
### To Launch the front end (React Js)
- cd frontend
-  npm install
- npm start
### To start the server (Node Js)
- cd backend
- npm install
- npm test
